Favicons

The favicon package code, located in the <head> tag of a HTML page, instructs different browsers and OSs where which icons to display in different uses.

Verifying favicons

It is important to verify that the favicons are correctly refenced on each individual page.

Use a service like RealFaviconGenerator's Favicon Checker to validate. Be sure to click the "favicons are not in root" option to reduce the number of false errors.
